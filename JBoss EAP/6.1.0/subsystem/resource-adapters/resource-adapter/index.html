JBoss EAP 6.1.0 Model Reference

  1. JBoss EAP 6.1.0
  2. subsystem=resource-adapters
  3. resource-adapter

The configuration of a resource adapter.

Children (3)

  • admin-objects The configuration of this resource adapter's admin objects.
  • config-properties Custom defined config properties.
  • connection-definitions The configuration of this resource adapter's connection definitions.

Attributes (13)

  • archive Specifies the resource adapter archive.
    • More Info
  • beanvalidationgroups Specifies the bean validation groups that should be used.
    • More Info
  • bootstrap-context Specifies the unique name of the bootstrap context that should be used.
    • More Info
  • config-properties Custom defined config properties.
    • More Info
  • module Specifies the module from which resource adapter will be loaded
    • More Info
  • transaction-support Specifies the transaction support level of the resource adapter.
    • More Info
  • wm-security Toggle on/off wm.security for this resource-adapter. In case of false all wm-security-* parameters are ignored, even thei defaults.
    • More Info
  • wm-security-default-groups Defines a default groups list that should be added to the used Subject instance
    • More Info
  • wm-security-default-principal Defines a default principal name that should be added to the used Subject instance
    • More Info
  • wm-security-domain Defines the name of the security domain that should be used
    • More Info
  • wm-security-mapping-groups List of groups mappings
    • More Info
  • wm-security-mapping-required Defines if a mapping is required for security credentials. A value of false means Case 1 as defined in section 16.4.3, and a value of true means Case 2 as defined in section 16.4.4.
    • More Info
  • wm-security-mapping-users List of user mappings
    • More Info
